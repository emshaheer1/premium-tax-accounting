## Run locally

1. Open the project folder.
2. Serve the root (e.g. with Live Server in VS Code, or `npx serve .`).
3. Open `index.html` via the server URL (e.g. `http://localhost:3000`).

No `npm install` or build step needed.

## Customize

- **Calendly:** Replace the “Book an Appointment” link (`https://calendly.com`) in `index.html` with your Calendly booking URL.
- **Logo:** Add your logo image and reference it in the header (e.g. inside the `.logo` link).
- **Colors/fonts:** Edit CSS variables in `assets/css/styles.css` (`:root`).

## Tech

- **HTML5** — Semantic sections, accessibility (ARIA, labels).
- **CSS** — Custom properties, Flexbox/Grid, scroll-triggered animations, hover/transition effects.
- **JS** — Sticky header on scroll, mobile menu, services dropdown, Intersection Observer for reveal animations, smooth scroll for anchor links.
